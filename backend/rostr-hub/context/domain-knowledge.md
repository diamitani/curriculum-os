# Domain Knowledge: Learning Science Principles

## How People Learn
1. **Spaced repetition** — Revisit concepts at increasing intervals
2. **Active recall** — Practice retrieving knowledge, not re-reading
3. **Interleaving** — Mix related topics to build discrimination
4. **Prerequisite scaffolding** — Foundational concepts before complex ones
5. **Project-based learning** — Apply knowledge in realistic contexts
6. **Immediate feedback** — Correct errors as they happen
7. **Dual coding** — Combine verbal and visual representations
8. **Chunking** — Group information into meaningful units

## Resource Quality Signals
| Signal | Indicates |
|--------|-----------|
| Peer-reviewed | Theoretical rigor, accepted knowledge |
| Author credentials | Domain authority, practical experience |
| Recency | Active field, evolving knowledge |
| Citations | Community validation, building on prior work |
| Platform reputation | Curated quality, structured curriculum |
| User ratings | Real-world effectiveness, accessibility |

## Curriculum Design Patterns
- **Block schedule**: Sequential modules of equal depth
- **Spiral curriculum**: Revisit topics at increasing complexity
- **Mastery learning**: Advance only when demonstrating competency
- **Flipped classroom**: Theory self-study + applied practice sessions
- **Bootcamp**: Immersive, time-boxed, project-driven
